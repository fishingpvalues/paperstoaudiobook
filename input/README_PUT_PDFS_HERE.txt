================================================================================
                            INPUT FOLDER
================================================================================

This is where you put your PDF files!

HOW TO USE:
-----------
1. Copy or move your PDF files into this folder
2. Run the converter (double-click run.bat or run.sh)
3. Your audiobooks will appear in the "output" folder

WHAT PDFs WORK BEST:
--------------------
✓ Scientific papers
✓ Research articles
✓ Study materials
✓ Textbooks (chapter PDFs)
✓ Technical documentation
✓ Any PDF with selectable text

TIPS:
-----
- Multiple PDFs? No problem! It processes them all
- Each PDF becomes one MP3 audiobook
- Keep original filenames - they're used for the MP3 names
- Typical conversion: 1-5 minutes per PDF on CPU

EXAMPLE:
--------
Input folder:
  - quantum_physics_chapter1.pdf
  - biology_notes.pdf
  - chemistry_paper.pdf

Output folder will have:
  - quantum_physics_chapter1.mp3
  - biology_notes.mp3
  - chemistry_paper.mp3
